<p>Произрастает в <a href="/wiki/%D0%A1%D0%B5%D0%B2%D0%B5%D1%80%D0%BD%D0%B0%D1%8F_%D0%90%D1%84%D1%80%D0%B8%D0%BA%D0%B0" title="Северная Африка">Северной Африке</a>, <a href="/wiki/%D0%92%D0%BE%D1%81%D1%82%D0%BE%D1%87%D0%BD%D0%B0%D1%8F_%D0%90%D0%B7%D0%B8%D1%8F" title="Восточная Азия">Восточной Азии</a>, <a href="/wiki/%D0%A2%D1%83%D1%80%D0%BA%D0%BC%D0%B5%D0%BD%D0%B8%D1%8F" title="Туркмения">Туркмении</a>, на <a href="/wiki/%D0%9A%D0%B0%D0%B2%D0%BA%D0%B0%D0%B7" title="Кавказ">Кавказе</a> и практически на всей территории <a href="/wiki/%D0%95%D0%B2%D1%80%D0%BE%D0%BF%D0%B0" title="Европа">Европы</a>. В <a href="/wiki/%D0%A0%D0%BE%D1%81%D1%81%D0%B8%D1%8F" title="Россия">России</a> встречается на <a href="/wiki/%D0%A1%D0%B5%D0%B2%D0%B5%D1%80%D0%BD%D1%8B%D0%B9_%D0%9A%D0%B0%D0%B2%D0%BA%D0%B0%D0%B7" title="Северный Кавказ">Северном Кавказе</a><sup id="cite_ref-GRIN_2-0" class="reference"><a href="#cite_note-GRIN-2">&#91;2&#93;</a></sup> и в Крыму.
</p><p>Растёт в пойменных лугах, по берегам рек.
</p>
<h2><span id=".D0.91.D0.BE.D1.82.D0.B0.D0.BD.D0.B8.D1.87.D0.B5.D1.81.D0.BA.D0.BE.D0.B5_.D0.BE.D0.BF.D0.B8.D1.81.D0.B0.D0.BD.D0.B8.D0.B5"></span><span class="mw-headline" id="Ботаническое_описание">Ботаническое описание</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%D0%9C%D1%8F%D1%82%D0%B0_%D0%B1%D0%BE%D0%BB%D0%BE%D1%82%D0%BD%D0%B0%D1%8F&amp;veaction=edit&amp;section=2" class="mw-editsection-visualeditor" title="Редактировать раздел «Ботаническое описание»">править</a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%9C%D1%8F%D1%82%D0%B0_%D0%B1%D0%BE%D0%BB%D0%BE%D1%82%D0%BD%D0%B0%D1%8F&amp;action=edit&amp;section=2" title="Редактировать раздел «Ботаническое описание»">править код</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/wiki/%D0%9C%D0%BD%D0%BE%D0%B3%D0%BE%D0%BB%D0%B5%D1%82%D0%BD%D0%B5%D0%B5_%D1%80%D0%B0%D1%81%D1%82%D0%B5%D0%BD%D0%B8%D0%B5" class="mw-redirect" title="Многолетнее растение">Многолетнее растение</a> высотой 20—60&#160;см.
</p><p><a href="/wiki/%D0%A1%D1%82%D0%B5%D0%B1%D0%B5%D0%BB%D1%8C" title="Стебель">Стебель</a> разветвлённый, рассеянно-волосистый.
</p><p><a href="/wiki/%D0%9B%D0%B8%D1%81%D1%82" title="Лист">Листья</a> <a href="/wiki/%D0%A7%D0%B5%D1%80%D0%B5%D1%88%D0%BE%D0%BA" class="mw-redirect" title="Черешок">черешковые</a>, эллиптические или продолговато-яйцевидные, листовая пластинка по краю тупозарубчато-пильчатая, у основания клиновидная.
</p><p><a href="/wiki/%D0%A6%D0%B2%D0%B5%D1%82%D0%BE%D0%BA" title="Цветок">Цветки</a> собраны в густые, почти шаровидные кольца; <a href="/wiki/%D0%A7%D0%B0%D1%88%D0%B5%D1%87%D0%BA%D0%B0" title="Чашечка">чашечка</a> с треугольными зубцами верхней губы и шиловидными нижней губы; <a href="/wiki/%D0%92%D0%B5%D0%BD%D1%87%D0%B8%D0%BA" title="Венчик">венчик</a> розово-лиловый с белой трубочкой.
</p><p><a href="/wiki/%D0%9F%D0%BB%D0%BE%D0%B4" title="Плод">Плод</a>&#160;— яйцевидный, бурый, блестящий, мелко сетчато-ямчатый <a href="/wiki/%D0%9E%D1%80%D0%B5%D1%88%D0%B5%D0%BA_(%D0%BF%D0%BB%D0%BE%D0%B4)" title="Орешек (плод)">орешек</a>.
</p><p>Цветёт в июле—августе. Плоды созревают в августе—сентябре.
</p>
<h2><span id=".D0.9F.D1.80.D0.B8.D0.BC.D0.B5.D0.BD.D0.B5.D0.BD.D0.B8.D0.B5"></span><span class="mw-headline" id="Применение">Применение</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%D0%9C%D1%8F%D1%82%D0%B0_%D0%B1%D0%BE%D0%BB%D0%BE%D1%82%D0%BD%D0%B0%D1%8F&amp;veaction=edit&amp;section=3" class="mw-editsection-visualeditor" title="Редактировать раздел «Применение»">править</a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%9C%D1%8F%D1%82%D0%B0_%D0%B1%D0%BE%D0%BB%D0%BE%D1%82%D0%BD%D0%B0%D1%8F&amp;action=edit&amp;section=3" title="Редактировать раздел «Применение»">править код</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>Сухие листья содержат до 2&#160;% <a href="/wiki/%D0%AD%D1%84%D0%B8%D1%80%D0%BD%D0%BE%D0%B5_%D0%BC%D0%B0%D1%81%D0%BB%D0%BE" class="mw-redirect" title="Эфирное масло">эфирного масла</a>, в состав которого входит до 95&#160;% <a href="/wiki/%D0%9F%D1%83%D0%BB%D0%B5%D0%B3%D0%BE%D0%BD" title="Пулегон">пулегона</a>, также <a href="/wiki/%D0%9C%D0%B5%D0%BD%D1%82%D0%BE%D0%BB" title="Ментол">ментол</a>, <a href="/wiki/%D0%9B%D0%B8%D0%BC%D0%BE%D0%BD%D0%B5%D0%BD" title="Лимонен">лимонен</a>, <a href="/wiki/%D0%94%D0%B8%D0%BF%D0%B5%D0%BD%D1%82%D0%B5%D0%BD" class="mw-redirect" title="Дипентен">дипентен</a>.
</p><p>Листья и верхушки цветущих стеблей употребляют как <a href="/wiki/%D0%9F%D1%80%D1%8F%D0%BD%D0%BE%D1%81%D1%82%D1%8C" class="mw-redirect" title="Пряность">пряность</a>.
</p><p>Эфирное масло используют в парфюмерии, консервном и кондитерском производстве, для ароматизации напитков, чая, уксуса.
</p><p>В народной медицине надземную часть растения применяли как <a href="/w/index.php?title=%D0%A0%D0%B0%D0%BD%D0%BE%D0%B7%D0%B0%D0%B6%D0%B8%D0%B2%D0%BB%D1%8F%D1%8E%D1%89%D0%B5%D0%B5&amp;action=edit&amp;redlink=1" class="new" title="Ранозаживляющее (страница отсутствует)">ранозаживляющее</a>, <a href="/wiki/%D0%90%D0%B1%D0%BE%D1%80%D1%82" title="Аборт">абортивное</a> и <a href="/wiki/%D0%90%D0%BD%D1%82%D0%B8%D1%81%D0%B5%D0%BF%D1%82%D0%B8%D0%BA" class="mw-redirect" title="Антисептик">антисептическое</a> средство, а также при <a href="/wiki/%D0%9A%D0%BE%D0%BA%D0%BB%D1%8E%D1%88" title="Коклюш">коклюше</a>, <a href="/wiki/%D0%91%D1%80%D0%BE%D0%BD%D1%85%D0%B8%D0%B0%D0%BB%D1%8C%D0%BD%D0%B0%D1%8F_%D0%B0%D1%81%D1%82%D0%BC%D0%B0" title="Бронхиальная астма">бронхиальной астме</a>, <a href="/wiki/%D0%98%D1%81%D1%82%D0%B5%D1%80%D0%B8%D1%8F" title="Истерия">истерии</a>.
</p>
